# Product and Category Management

Этот проект реализует функциональность для управления продуктами и категориями продуктов. Он включает в себя классы Product и Category, которые позволяют создавать, хранить и управлять информацией о продуктах и их категориях.

## Описание классов

### Product
Класс Product представляет продукт и включает следующие атрибуты:

- name: Название продукта (строка).
- description: Описание продукта (строка).
- price: Цена продукта (число с плавающей точкой).
- quantity: Количество продукта на складе (целое число).

### Category
Класс Category представляет категорию продуктов и включает следующие атрибуты:

- name: Название категории (строка).
- description: Описание категории (строка).
- products: Список объектов класса Product, принадлежащих данной категории.

Класс также отслеживает общее количество созданных продуктов и категорий через статические переменные:

- product_count: Общее количество созданных продуктов.
- category_count: Общее количество созданных категорий.
